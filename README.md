# DART-OT

**DART-OT**는 DART 공시 API를 기반으로 정기보고서의 차입금 관련 주석을 수집하고, 차입금·사채 내역의 이자비용이 합리적으로 계상되었는지 검토하는 **이자율 오버롤 테스트 엑셀 파일 생성 도구**입니다.

감사·검토 업무에서 반복적으로 수행되는 공시 조회, 차입금 내역 필터링, 주석 특이사항 정리, 이자율 적정성 검토를 하나의 데스크톱 GUI 애플리케이션에서 처리할 수 있도록 만드는 것을 목표로 합니다.

## 문제 상황

차입금 및 사채의 이자비용 검토는 회사의 재무제표 주석, 정기보고서, 차입 조건, 평균 이자율, 기중 변동 내역 등을 함께 확인해야 하는 작업입니다. 그러나 실무에서는 다음과 같은 비효율이 자주 발생합니다.

- DART에서 여러 연도의 정기보고서를 수동으로 검색하고 다운로드해야 합니다.
- 보고서마다 차입금 주석의 위치와 표현 방식이 달라 필요한 정보를 찾는 데 시간이 많이 걸립니다.
- 차입금, 사채, 유동성 대체 등 관련 내용을 사람이 직접 선별해야 합니다.
- 이자비용이 평균 차입잔액과 공시 이자율에 비추어 적절한지 별도로 계산해야 합니다.
- 여러 회사 또는 여러 기간을 검토할 때 동일한 확인 절차가 반복됩니다.

이러한 수작업 중심의 검토 방식은 시간이 많이 소요되고, 누락·전사 오류·검토 기준 불일치가 발생할 가능성이 있습니다.

## 해결 방안

DART-OT는 DART API를 활용해 최근 10년치 정기보고서를 자동으로 수집하고, 차입금 관련 공시 내용을 필터링한 뒤, 이자율 오버롤 테스트 결과를 엑셀 형태로 정리합니다.

사용자는 회사명 또는 종목코드 등 필요한 조건을 입력하고 실행하면, 프로그램이 정기보고서 목록 수집부터 차입금 주석 추출, 특이사항 정리, 이자율 적정성 테스트까지 일관된 흐름으로 처리합니다.

## 주요 기능

- DART 공시 API를 통한 최근 10년치 정기보고서 조회
- 회사명, 보고서명, 접수일, 접수번호, 종목코드 정리
- 정기보고서 내 차입금·사채 관련 주석 필터링
- 차입금 주석공시 내용 및 관련 특이사항 요약
- 공시 이자율과 계산된 이자비용을 비교하는 이자율 오버롤 테스트
- 오차범위 ±5% 기준에 따른 적정 계상 여부 판단
- `_start.bat` 또는 `DART-OT.exe`를 통한 원클릭 실행
- 데스크톱 GUI에서 회사 검색, 회사 선택, 엑셀 파일 생성

## 엑셀 산출물 구성

프로그램 실행 결과는 검토 목적에 맞게 다음과 같은 시트로 정리됩니다.

### 1. 정기보고서 목록

최근 10년치 DART 정기보고서 목록을 표시합니다.

- 회사명
- 보고서명
- 접수일
- 접수번호
- 종목코드

### 2. 차입금 필터링 내역

정기보고서에서 차입금 및 사채와 관련된 공시 내용을 별도로 추출해 표시합니다.

- 차입금 관련 주석공시 내용
- 사채 및 차입 조건 관련 정보
- 유동성 대체, 만기, 금리 조건 등 특이사항
- 검토 시 주의가 필요한 문구 또는 이슈

### 3. 이자비용 필터링 내역

정기보고서의 금융원가 주석에서 이자비용 검토에 사용할 수 있는 차입·금융부채 관련 이자비용 항목을 별도로 추출해 표시합니다.

- 이자비용(금융원가)
- 상각후원가 측정 금융부채 이자비용
- 기타 금융부채 이자비용
- 차입금 이자비용 및 사채 이자비용

### 4. 이자율 오버롤 테스트

차입금 내역을 기반으로 이자비용이 적절히 계상되었는지 확인합니다.

- 차입금 잔액 및 적용 이자율
- 기대 이자비용 계산
- 실제 계상 이자비용과의 차이 분석
- 오차율 산정
- ±5% 오차범위 기준 적정 여부 판단

## 프로젝트 필요성

DART-OT는 단순한 공시 조회 도구가 아니라, 감사·회계 검토 업무에서 반복되는 차입금 이자율 검토 절차를 표준화하기 위한 도구입니다.

이 프로젝트를 통해 기대할 수 있는 효과는 다음과 같습니다.

- 반복적인 DART 조회 업무 시간 단축
- 차입금 관련 공시 누락 가능성 감소
- 이자율 오버롤 테스트 기준의 일관성 확보
- 검토 결과를 엑셀 산출물로 보관해 리뷰 및 문서화 용이
- 회사별·기간별 비교 검토 효율 향상
- GUI 기반 실행으로 비개발자도 쉽게 사용 가능

## 실행 방식

최종 애플리케이션은 사용자가 별도의 복잡한 명령어를 입력하지 않아도 실행할 수 있도록 구성합니다.

실행 방식:

```bat
_start.bat
```

또는 Windows 실행 파일을 직접 실행할 수 있습니다.

```bat
DART-OT.exe
```

`_start.bat` 또는 `DART-OT.exe` 실행 시 데스크톱 프로그램 창이 열리고, 사용자는 GUI에서 회사 정보 입력, 회사 검색 및 선택, DART 공시 조회, 차입금 필터링, 이자율 오버롤 테스트 결과 생성 작업을 진행할 수 있습니다.

공시 목록과 원문을 먼저 확인하고 싶은 경우에는 별도 웹 뷰어를 실행할 수 있습니다.

```bat
_disclosure_viewer.bat
```

또는 다음 실행 파일을 직접 실행할 수 있습니다.

```bat
DART-Disclosure-Viewer.exe
```

공시 뷰어는 브라우저에서 회사별 DART 공시 목록을 조회하고, 각 공시의 DART 원문 링크 및 앱 내부 원문 텍스트 검색 기능을 제공합니다.

## 사용 전 준비사항

DART 공시 조회를 위해 OpenDART API 인증키가 필요합니다. 실행 화면에서 API 키와 회사명 또는 종목코드를 입력한 뒤 `회사 검색`을 누르면 후보 회사 목록이 표시됩니다. 목록에서 정확한 회사를 선택한 뒤 `엑셀 파일 생성`을 누르면 정기보고서 수집을 시작합니다.

API 키를 매번 입력하지 않으려면 실행 화면의 `API 키 저장`을 선택하면 됩니다. 저장된 키는 로컬 설정 파일(`.dart_ot_config.json`)에만 보관되며, GitHub에는 업로드되지 않도록 제외되어 있습니다.

## 현재 구현 범위

현재 버전은 DART API에서 정기보고서 목록과 원문 문서를 수집하고, 차입금·사채·금융부채·이자율·금리·가중평균·이자비용·금융원가 등 관련 키워드를 기준으로 원문 라인을 필터링합니다. 이자율 오버롤 테스트는 주석에서 검출한 개별 차입금·사채 이자율을 평균해 `검출평균이자율`로 표시하고, 이를 바탕으로 산정한 예상 이자비용을 회사가 계상한 이자비용과 비교해 오차율이 ±5% 이내인지 판단합니다. 실제 이자비용은 금융원가 주석의 상각후원가 측정 금융부채 이자비용, 기타 금융부채 이자비용, 차입금 이자비용, 사채 이자비용 등 차입·금융부채 관련 이자비용을 우선 사용합니다. 회사가 별도의 평균 차입이자율을 공시한 경우에는 참고 비교값으로 활용하며, WACC 또는 가중평균자본비용은 참고 검출수로만 남깁니다. 전기 대비 금액 증감과 변동률도 함께 표시합니다. 금액 컬럼은 엑셀에서 천 단위 콤마가 표시되도록 숫자 서식이 적용됩니다.

전기 대비 검출금액합계가 30% 이상 변동하면 `주의요망`으로 표시하고, 별도 사유 열에 주의가 필요한 이유를 설명합니다.

생성된 엑셀 파일은 `outputs` 폴더에 저장되며, 실행 화면에서 결과 파일 또는 저장 폴더를 바로 열 수 있습니다.

## 별도 감가상각비 테스트 도구

금융비용 주석 구조 때문에 이자비용 검토가 불안정한 경우를 대비해, 별도 프로그램인 **DART-DEP**도 함께 제공합니다. DART-DEP는 기존 DART-OT를 대체하지 않고, 감가상각비 및 상각비 검토를 위한 별도 산출물을 생성합니다.

실행 방식:

```bat
_depreciation_start.bat
```

현재 DART-DEP 초안은 정기보고서에서 다음 주석을 구분해 엑셀로 정리합니다.

- `유형자산`, `무형자산`, `사용권자산`, `투자부동산` 주석
- `비용의 성격별 분류`, `판매비와관리비`, `매출원가` 주석의 감가상각비 및 상각비 항목
- 자산 주석 필터링, 감가상각비 필터링, 감가상각비 오버롤 요약 시트

감가상각비 오버롤 시트는 우선 주석에서 감가상각비·상각비 항목이 식별되는지와 금액 후보를 정리하는 구조이며, 내용연수와 평균 장부금액을 이용한 기대 감가상각비 산정 로직은 후속 단계에서 확장할 수 있습니다.
